TFProfiler

TFProfiler is an app that aims to profile TensorFlow Lite model and measure its performance using FPS, model initialization time, model inference time, memory consumption, etc. You can tweak model runs with different delegates (CPU, GPU, NNAPI, HEXAGON), XNNPACK option, number of threads.

Eye State classification

The eye state classifier is based on transfer learning technique. I use transfer learning where we take a pre-trained network (trained on about a million general images), use it to extract features, and train a new layer on top for our own task of classifying images of eyes.
